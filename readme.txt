Como mostrado na apresentação, não existem routings em botões, pelo que a navegação tem de ser feita
maioritariamente mudando o url

Página de login (starting page)
http://localhost:4200/login

Página de SignUp
http://localhost:4200/signup

Página de um user
http://localhost:4200/user

Página resultado de um search
http://localhost:4200/searchresults

Página de MyClans
http://localhost:4200/myclans

Página para Criar um Clan
http://localhost:4200/createclan

Homepage
http://localhost:4200/home

Página de Post
http://localhost:4200/postpage

Página para editar o perfil
http://localhost:4200/editprofile

Página de um Clan
http://localhost:4200/clan

Página de erro 404
http://localhost:4200/page-not-found (ou outro url qualquer não previsto)
